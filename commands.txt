~ (не команда) - рабочая директория (КОРНЕВАЯ ПАПКА) текущего пользователя
c:/Users/ibook

cd - change directory 
ls - show list files and directories in current directory
cd .. - перейти на уровень выше
pwd - отопражает путь до текущей папки
"tab" - автоматически добавляет фразу(название папки)


______
Команды git

git init - создаёт гит-репозиторий
git add  - добавляет указанный файл в гит-репозиторий 
git add . - добавляет все файлы в папки

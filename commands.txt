cd - change directory

cd .. - перейти на уровень выше
pwd - отопражает путь до текущей папки
"tab" - автоматически добавляет фразу(название папки)

Команды git

git init - создаёт гит-репозиторий
git add  - добавляет указанный файл в гит-репозиторий 
git add . - добавляет все файлы в гит-репозиторий 
git status - отображает статус всех файлов в текущем репозитории (помечены красным - не отслеживаются; зеленым - отслеживаются и добавлены в репозиторий)
git commit -m "название для контрольной точки" - создание коммита

git remote add origin https://github.com/ymartsevich/For-git-commands.git - эта команда вызывается единожды для связи локального репозитория и удаленного репозитория
 
git push origin master - эта команда толкает все наши коммиты(изменения) на удаленный сервер
